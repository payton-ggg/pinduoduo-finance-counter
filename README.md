# 📊 Pinduoduo Finance Counter

> **Профессиональная экосистема для финансового контроля и складского учета товаров с маркетплейса Pinduoduo.**

Этот инструмент создан для предпринимателей, которые ценят точность и хотят видеть реальную прибыльность каждой единицы товара. Больше никаких сложных таблиц в Excel — всё управление в одном современном веб-интерфейсе.

---

## 🚀 Ключевые возможности

### 💰 Умная экономика
- **Авто-конвертация валют**: Мгновенный расчет себестоимости на основе актуального курса **CNY → UAH**.
- **Учет всех затрат**: Система учитывает не только цену закупки, но и стоимость доставки в Украину, а также комиссию за управление (Management Fee).
- **Расчет чистой прибыли**: Автоматическое сопоставление доходов и расходов по каждому артикулу.

### 📦 Продвинутый складской учет
- **Инвентаризация 2.0**: Отслеживание закупленного количества, проданных единиц и текущих остатков.
- **Архивация**: Возможность скрывать неактуальные позиции, сохраняя их в истории для анализа.
- **Интеграция с Cloudinary**: Молниеносная загрузка и хранение фото товаров в облаке.

### 🎧 Специализация на электронике
- **Детальные пресеты**: Поля для технических характеристик (чипы Airoha/Huilian, датчики уха, беспроводная зарядка, гироскоп и т.д.).
- **Кастомные поля**: Возможность указывать комплектацию и вес каждой позиции.

## ⚙️ Установка и настройка

### 1. Подготовка окружения
Клонируйте проект и установите зависимости:
```bash
git clone https://github.com/payton-ggg/pinduoduo-finance-counter.git
cd pinduoduo-finance-counter
npm install
```

### 2. Конфигурация `.env`
Создайте файл `.env` и заполните его следующими данными:
```env
# Database
DATABASE_URL="postgresql://..."

# Cloudinary
CLOUDINARY_CLOUD_NAME="..."
CLOUDINARY_API_KEY="..."
CLOUDINARY_API_SECRET="..."
```

### 3. База данных
Сгенерируйте клиент Prisma и синхронизируйте схему:
```bash
npx prisma generate
npx prisma db push
```

### 4. Запуск
```bash
npm run dev
```
